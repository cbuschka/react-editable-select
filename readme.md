# reactstrap-editable-select

### A select component which can switch to text input to allow adding new elements.

## Prerequesites
* reactstrap
* bootstrap
* font-awesome
* yarn
* webpack
* es6

## Try it out
* Activate nodejs
```
nvm use
```

* Install yarn
```
npm -g install yarn
```

* Start webpack dev server
```
yarn start
```

* Goto http://localhost:3001/
